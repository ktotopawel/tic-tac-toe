
# Tic-Tac-Toe

This is a project made as a part of theodinproject.com curriculum. </br>
A simple tic-tac-toe game for two players.


## Features

- Ability to change player names
- Ability to play back to back with a 1.5 second reset timer
- Score counter
- Reset button
